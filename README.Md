# HR Analytics Dashboard

## Project Overview

This project analyzes employee attrition patterns using HR workforce data and presents key business insights through an interactive dashboard built in LibreOffice Calc.

The objective is to identify factors influencing employee turnover and provide actionable insights to improve workforce retention strategies.

## Business Insights

* Overall employee attrition rate stands at **16.1%**.
* The **Sales department** records the highest attrition rate (**21.6%**).
* Employees working **overtime** are nearly **3× more likely** to leave the organization compared to employees not working overtime.
* **Sales Representatives** exhibit the highest attrition rate (**39.8%**) among all job roles.
* Employees aged **18–25 years** experience the highest attrition rate (**35.8%**).
* Employees earning below **3,000/month** show the highest attrition rate (**28.6%**).
* Workload, compensation, career stage, and job role emerge as the primary drivers of employee turnover.
